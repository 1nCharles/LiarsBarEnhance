# 功能

### 被动技

 - 移除视角转动限制
 - 修复中文玩家名显示`□`
 - 移除玩家名字长度限制（HUD与聊天）
 - 移除只能发送小写限制
 - 移除敏感词限制（房主需要）

### 主动技

 - 按I疯狂转头
 - 按住O张嘴
 - ↑↓←→控制头的移动
 - WASD控制身体移动
 - 按住鼠标右键转动控制身体转动

### 做不到

 - 看别人的牌
 - 修改手中的牌型
 - 不死

# 安装

 1. 安装[BepInEx5](https://github.com/BepInEx/BepInEx/tree/v5-lts)（教程自己找）
 2. 从[Release](https://github.com/dogdie233/LiarsBarEnhance/Release)下载适用于你的游戏版本的文件（dll文件）
 3. 将下载的dll放置在插件文件夹
